# Diagonal Duel – Blazor WebAssembly Game

**Diagonal Duel** is a web-based game built using Blazor WebAssembly. The player is presented with a randomly generated square matrix 
and must guess the absolute difference between the sums of its primary and secondary diagonals.

## Features
- Random matrix generation for each round
- Score tracking over 5 rounds
- Real-time input validation and feedback
- Built with C# and Blazor WebAssembly (.NET 8)

## How It Works
- The game displays a square matrix with random values.
- The player enters a guess for the absolute diagonal difference.
- Points are awarded based on the accuracy of the guess.

## Technologies Used
- .NET 8
- Blazor WebAssembly
- C#
- Razor Components
- Bootstrap (for basic UI styling)
